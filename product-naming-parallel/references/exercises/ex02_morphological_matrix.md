# Øvelse 2: Morphological Matrix
Minimum: 20 kombinationer

Byg tabellen fra briefen:

| Akse | Udfyld fra brief |
|------|-----------------|
| Kernehandling | hvad produktet gør bogstaveligt |
| Menneskelig | Bob, Finn, Otto, Hugo, Vera, Arlo, Milo, Rex, Cleo, Kai |
| Sanselig | tick, hum, drift, grain, pulse, spark, flint, hatch, glow |
| Kategori-modsætning | hvad kategorien ALDRIG ville kalde sig |

Kombinér systematisk på tværs af alle akser.
WelcomeBob = [Welcome] × [Bob]. TickTick = [Tick] × [Tick].

Valg-kriterium: videreføres kun hvis BEGGE led er ladede i sig selv.
Ladet = varme, energi eller præcis mening alene.
Welcome er ladet. Bob er ladet. Join er ikke ladet. Sit er ikke ladet.

Skriv alle 20+ kombinationer ud.
