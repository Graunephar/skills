# Øvelse 1: Iterativ Udvidelse
Minimum: 25 kandidater

Kør i 4 lag:

**Lag 1:** Identificér 5-8 kerneord fra briefen. Skriv dem ud.

**Lag 2:** For hvert kerneord: find 6-8 synonymer, hverdagsudtryk, nabobegreber.

**Lag 3:** For hvert udvidet ord: find kombinationspartnere.
Tænk: steder, tidspunkter, handlinger, præpositioner, mennesker.

**Lag 4:** Smelt kombinationer til ét ord eller kort sammensætning.

Eksempel:
```
Kerneord: mødes
→ meet, greet, catch up, run into, "in town", "stopping by"
→ "in town" + hvornår → "intown thursday" → Intown
→ "stopping by" smeltet → Stopby
→ greet + her → GreetHere
```

Skriv ALLE kandidater ud. Ingen filtrering.
