# Øvelse 8: Random Stimulus (De Bono)
Minimum: 6 kandidater (2 per stimulus)

Vælg 3 tilfældige substantiver — ét naturord, ét håndværksord, ét madord.

For hvert:
1. List 5 egenskaber ved ordet
2. Tving forbindelser til produktet
3. Generér mindst 2 kandidater
4. Stil: "Hvad er det mest uventede reelle kandidat fra dette?"

Eksempel:
```
Ord: Mørtel
Egenskaber: binder, midlertidigt flydende, permanent, fuger, håndværk
Forbindelser → binder to mennesker midlertidigt → Set? Mortar? Bond?
Mest uventede: Set (mørtel der sætter sig = mødet der bliver til noget)
```
