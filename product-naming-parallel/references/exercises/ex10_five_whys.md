# Øvelse 10: Five Whys → Navngiv roden
Minimum: 5 kandidater

Spørg "hvorfor" 5 gange for at komme til den dybeste sandhed om produktet.
Navngiv derefter roden — ikke overfladen.

Format:
```
Overfladen: [hvad produktet gør]
Hvorfor er det værdifuldt? → ...
Hvorfor er det et problem det ikke eksisterer? → ...
Hvorfor opstår problemet? → ...
Hvorfor er det ikke løst allerede? → ...
Dybeste sandhed: [roden]

Navne fra roden: [kandidater]
```

Navne fra rodkonceptet er fundamentalt anderledes end navne fra overfladen.
