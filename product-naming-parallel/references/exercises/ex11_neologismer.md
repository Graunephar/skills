# Øvelse 11: Neologismer og Portmanteau
Minimum: 15 konstruerede ord

Tre metoder:

**Portmanteau** — smelt to ladede ord:
```
[ord A] + [ord B] → find overlap i lyd → smeltet ord
welcome + via → Welvio
travel + fellow → Trellov
```

**Rodkombination** — latinsk/græsk rødder:
```
com- (sammen) + via (vej) → Comvia
syn- (samme) + loc (sted) → Synlo
trans- (igennem) + iter (rejse) → Transiter → Transit
```

**Suffiksering** — ladet ord + suffix der giver liv:
```
[ladet ord] + -io, -o, -en, -al, -er
greet + -io → Greeto
layer + -o → Layero? Nope. → Layro?
```

Krav: hvert konstrueret ord skal:
1. Kunne udtales flydende
2. Have mindst ét ladet led
3. Ikke lyde som tilfældige bogstaver
