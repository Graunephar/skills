# OUTPUT — Præsentér alle der overlever, fremhæv 2

Præsentér **alle navne der bestod alle 5 filtre** — ikke kun 3.

Format per navn:
```
**[Navn]** — [én linje: hvorfra + hvad der gør det interessant]
```

Derefter fremhæv **præcis 3 favoritter** med fuld begrundelse:
```
⭐ **[Navn]**
Hvorfra: [øvelse + logik]
Styrke: [specifikt — ikke generelt]
Svaghed: [reel risiko]
I brug: "[naturlig sætning]"
```

De 3 favoritter skal være **maksimalt forskellige fra hinanden** — ikke to af samme type.

Afslut med ét spørgsmål fra `references/feedback.md`.
